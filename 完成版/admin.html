📖 使い方マニュアル — 管理画面
📋 スクリプトタブ
  • カテゴリ名・サブカテゴリ名はダブルクリックで編集できます（フォーカスアウトで確定）。
  • 「＋ カテゴリを追加」でカテゴリを追加できます（色はランダムで付与）。
  • 「⠿」ハンドルをドラッグしてスクリプトの並べ替えができます。
✉️ メールタブ
  • カテゴリ・サブカテゴリ・テンプレートの追加・編集・削除ができます。
  • カテゴリ名はダブルクリックで編集できます（フォーカスアウトで確定）。
  • 「⠿」ハンドルをドラッグしてテンプレートの並べ替えができます。
🖥 画面遷移タブ
  • 「＋ パターンを追加」でパターンを作成します。パターン名はダブルクリックで編集できます。
  • 「⠿」ハンドルをドラッグしてパターン・画面の並べ替えができます。
  • 📊 Excelインポート（サイドバー見出し右）または Excel ファイルの D&D で、シートごとにパターンを自動作成して画像を取り込めます。モーダル内でシート・画像ごとに選択できます。
  • 画像ファイルをドロップすると現在選択中のパターンに画像が追加されます。
🖼 画面設定（右サイドバー）
  • 画面名・メモを編集できます。メモは screen.html のプレビュー画面名下に表示されます。
  • スクリーンショットの差し替えが可能です（D&D または「📂 クリック」）。
  • 「📍 ホットスポット」モードで画面上をドラッグして遷移エリアを作成できます。
  • ホットスポットのメモは screen.html のプレビューでカーソルを乗せたときに表示されます。
📝 更新履歴タブ
  • 「＋ 更新履歴を追加」で先頭行に行が追加されます。
  • 各セルをダブルクリックで直接編集できます。
  • 「📅」ボタンのカレンダーから日付を選択できます。
💾 保存と自動エクスポート
  • 「💾 保存して反映」を押すと全データが保存され、同時に JSON ファイルが自動ダウンロードされます。
  • ⚠️ 未保存バッジが表示されている間は変更が本番画面に反映されていません。
  • ダウンロードした JSON を各ページ（index / mail / screen）にドロップすると即時反映されます。
📥 D&D インポート対応一覧
  • JSON ファイル：スクリプト・メール・画面遷移データを一括インポートします（確認あり）。
  • Excel (.xlsx)：埋め込み画像をシートごとにパターンとして取り込めます。
  • 画像ファイル：画面遷移タブが開いているときにドロップすると現在のパターンに追加されます。
