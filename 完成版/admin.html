📖 使い方マニュアル — 管理画面
📋 スクリプトタブ
  • カテゴリ名・サブカテゴリ名はダブルクリックで編集できます（フォーカスアウトで確定）。
  • 「＋ カテゴリを追加」でカテゴリを追加（色はランダム付与）。「⠿」ドラッグで並べ替え。
  • スクリプト本文・選択肢はエディタエリアで直接編集できます。
✉️ メールタブ
  • カテゴリ・サブカテゴリ・テンプレートの追加・編集・削除ができます。
  • カテゴリ名はダブルクリックで編集。「⠿」ドラッグでテンプレートの並べ替え。
🖥 画面遷移タブ — パターン・画面管理
  • 「＋ パターンを追加」でパターンを作成。パターン名はダブルクリックで編集。
  • 「⠿」ドラッグでパターン・画面の並べ替え。
  • 📊 Excelインポート（サイドバー見出し右）または Excel ファイルの D&D で、シートごとにパターンを自動作成。モーダル内でシート・画像ごとにチェックして選択追加できます。
  • 画像ファイルをドロップすると現在選択中のパターンに一括追加されます。
🖥 画面遷移タブ — 画面設定（右サイドバー）
  • 画面名・メモを編集できます。メモは screen.html プレビューの画面名下に表示されます。
  • スクリーンショットは D&D またはクリックで差し替え可能。リサイズ・トリミングも行えます。
  • トリミング：モードを切り替えると全体選択状態になります。ハンドル（8点）をドラッグして範囲を調整し「✂️ トリミング」で確定。幅・高さの入力欄に px を指定すると出力サイズを固定できます。
📍 ホットスポット
  • 「📍 ホットスポット」モードで画像上をドラッグして遷移エリアを作成します。
  • 遷移先タイプは 🖥 同パターン（同パターン内の別画面）・🗂 別パターン（別パターンの画面）・✉️ メール（メールテンプレート）の3種類から選択できます。
  • メモを設定すると screen.html プレビューでカーソルを乗せたときにツールチップ表示されます。
🖼 画像ライブラリタブ
  • 登録されている全画像を一覧表示します。サムネイルクリックで拡大プレビュー。
  • 画像名をクリックしてリネーム。タグ（#タグ名）を追加して検索バーでフィルタリングできます。
  • 「🔄 差し替え」で画像データを更新（使用中の全パターンに即反映）。
  • 🔀 内容で統合：画像データが同一のエントリを統合。🔗 同名を統合：名前が同じエントリを統合。🧹 未使用を削除：どの画面にも使われていない画像を一括削除。
📝 更新履歴タブ
  • 「＋ 更新履歴を追加」で先頭行に追加。各セルはダブルクリックで直接編集。
  • 「📅」ボタンのカレンダーから日付を選択できます。
💾 保存・エクスポート・インポート
  • 「💾 保存して反映」を押すと全データが保存され、JSON ファイルが自動ダウンロードされます（エクスポートサイズは MB 表示）。⚠️ 未保存バッジがある間は未反映です。
  • ダウンロードした JSON を index / mail / screen の各ページにドロップするとリロードなしで即時反映されます。
  • JSON ファイル D&D：スクリプト・メール・画面遷移を一括インポート（確認あり）。画像は自動的に画像ライブラリに移行されます。
  • Excel (.xlsx) D&D：埋め込み画像をシートごとにパターンとして取り込めます。
